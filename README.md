# npms
实用Npm包集合

## md2img
```bash
Usage: md2img [options] <input>

Convert Markdown to image with GitHub theme

Arguments:
  input                Input Markdown file

Options:
  -o, --output <path>  Output image path (default: <input>.png)
  -f, --format <fmt>   Output format: png | jpg (default: "png")
  -w, --width <px>     Page width in pixels (default: "800")
  -s, --scale <n>      Device scale factor (retina = 2) (default: "2")
  --theme <name>       Color theme: light | dark (default: "light")
  --quality <n>        JPEG quality 0-100 (default: "90")
  -h, --help           display help for command
```
